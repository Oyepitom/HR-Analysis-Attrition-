👥 HR Analysis — Employee Attrition

A data analysis project examining employee attrition patterns within an organisation, using Power BI to uncover the key factors driving staff turnover and inform HR decision-making.

📌 Project Overview

Employee attrition is a critical HR challenge for any organisation. This project analyses workforce data to identify which departments, roles, age groups, and satisfaction levels are most associated with employee exits — providing HR teams and management with actionable insights to improve retention.

🎯 Objectives

- Analyse employee attrition rates across departments and job roles
- Identify key demographic and job-related factors linked to turnover
- Visualise attrition trends using charts and dashboards
- Provide data-backed recommendations for improving employee retention

🛠️ Tools & Skills Used

| Tool           | Purpose |
| Power BI       | Interactive dashboard and visual storytelling |
| Power Query    | Data transformation and preparation |
| DAX            | Calculated attrition metrics and KPIs |

 📂 Files

| File | Description |
|------|-------------|
| HR Attrition dataset | Raw and cleaned employee data |

📈 Key Areas of Analysis

- Overall Attrition Rate — Percentage of employees who have left vs active workforce
- Department Breakdown — Which departments have the highest turnover
- Age & Gender Analysis — Attrition patterns by demographics
- Job Role & Satisfaction — Link between role type, satisfaction scores, and exits
- Tenure Analysis — How years at the company relate to attrition likelihood
- Salary & Performance — Whether compensation level correlates with turnover

 🔍 Key Insights

- Employees in certain departments and lower salary bands showed higher attrition rates
- Early-tenure employees (0–2 years) had significantly higher exit rates
- Job satisfaction and work-life balance scores were strong predictors of attrition
- Targeted retention strategies could reduce overall turnover rate

🚀 How to View

1. Download the `.pbix` file from this repository
2. Open in [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
3. Use slicers to explore attrition by department, role, and demographics
